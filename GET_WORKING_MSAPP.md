## WHY This is Necessary

.msapp format is proprietary and includes:
- Binary encoded metadata
- SHA256 checksums in Entropy/checksum.json
- Component GUIDs that must match across files
- Encrypted connection references
- Tool-specific metadata

Microsoft doesn't publish the full spec - it's generated by their tools only.
